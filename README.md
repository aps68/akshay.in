# Akshay P S - Animated CV

This is a professional, animated CV website built with HTML, CSS, and Vanilla JavaScript.

## content
The content has been extracted from `Akshay_Updated_Resume.ats.pdf`.

## How to View
1.  Navigate to the `cv_website` folder: `c:\trade\cv_website`
2.  Open `index.html` in your web browser (Chrome, Edge, Firefox).

## Features
-   **Animated Hero Section**: Glitch text effect and candlestick chart animation.
-   **Interactive Background**: Particle network that reacts to mouse movement.
-   **Scroll Animations**: Sections fade in and slide up as you scroll.
-   **Stats Counter**: Numbers count up when they come into view.
-   **Responsive Design**: Works on desktop and mobile.
